工夫

液肥濃度調整

管理方法

  • 満水でちょうど2Lになる容器を使用する。500mLおきに目盛りを振る。

  • 後述の通り、若い時期は6000等量（1200uS/cmの水 × 5L）で管理することとする。

  • 新規作成の場合、4gのハイポネックスを5L（1.6L×3本または2L×2本＋1L×1本）に溶かせば良い。

  • 追加の場合、電気伝導度と量を確認する。その等量を計算し、それを補うように追加する。

    • 追加用の液は、1.5Lで3000等量で調整しておく。0.5Lで1000等量。
    • 1.5Lの水を作り、そこに2g溶かした状態。

調整方法検討の過程

微粉ハイポネックスの指示に従って2Lに2gで作ったところ、電気伝導度は1900uS/cmだった。動画では1500um/cmと言っていたが……。

  • 2L容器だが、入っていたのはおよそ1.65Lだった。
    • とすると、本来の2Lに2gは、1560uS/cm。割とぴったり！
  • 水道水が200uS/mだった。ミネラルウォーターとかも同程度らしいので、ここが原因ではなさそうだが……。
  • 一般的には、1000uS/cmでも（イチゴなどは）過多らしい？
  • このサイトによれば、オクラの水耕栽培の電気伝導度は花がつく時期は2000uS/cmでは不足らしい。
    • AIによれば、「苗が若い時期は1.0〜1.2程度、収穫期（実がなる時期）は肥料を多く必要とするため1.5〜2.0程度に調整します」とある。
  • バケツには1.8L×2.8杯分の水が入った。すなわち、約5L。
  • とりあえず、若い時期は1200uS/cmでやってみよう。1200×5=6000等量の電気伝導度であればOK。
    • 2gを2Lで溶かすと1500×2=3000等量。ということは、5Lの水に4g溶かせば6000等量。

トラブルシューティング

発芽

オクラは土の温度が25～30℃くらいないと発芽しない。 春でそこそこ暖かくても、温まらない日陰では発芽しない。また、発芽しないままだと種が腐ってしまう。

最初、平均気温20℃、最低気温10℃くらいの時期に播種したがしばらく確認できず。

日なたに置いたからか、間もなく発芽が確認。
