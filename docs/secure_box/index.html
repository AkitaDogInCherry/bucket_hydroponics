プロジェクトボックスの固定

骨組み

あまり大げさにしない（単管ではなく塩ビを使用、径もあまり大きくしない）という観点から骨組みを選定。

VP-13の塩ビパイプは外径19mm。19mmの単管のクランプなどを使用することができる。

19mmのパイプは自転車やバイクのハンドルと似た径であり、後述のクランプを使用することができる。

プロジェクトボックスの固定

Uボルトなどを使うと安くできるが、ここではボックスに穴を開けず、取り外しも簡単になるようにしたかった。
